# +11/state Coverage Batch — Target Slots (the contract)

_Phase 2 + 3 deliverable. 11 target slots per state = **66**. Each slot defines a real user moment the
current 187-SIO corpus does **not** answer well (see `sio_coverage_batch_11_coverage_map.md`). No SIO is
authored unless it fills a slot here. **Source leads are unverified search targets** — the final quote is
whatever a real, caption-verified moment actually says (yt-dlp), not the lead's paraphrase. Drop any lead
that fails verification and find another. Honesty > hitting 11._

**Cross-cutting goals for the batch**
- Add **few/no** SIOs from over-used families (School of Greatness, Huberman Lab, Louisiana Channel) or
  speakers already at ≥2 (Huberman, Robbins, Brown, Grant, Clear, …).
- Lean into the thinnest register cell — **direct/challenging** — where it honestly fits (PP:2, ED:3, IT:3, MG:3, DC:4 today).
- Prefer ≥8 video-backed per state; widen speaker gender / culture / field / age / communication style.
- Do not worsen known magnets (IT: Shankar; ED: Brooks).

Legend for coverage verdicts on the 20 questions: 🟢 strong existing answer · 🟡 weak/tangential · 🔴 none.

## Selection discipline (applies to every slot)

1. **Caption-verified or it doesn't ship.** The lead is a search target; the SIO's quote is whatever the
   verified transcript says, at a real timestamp. If no honest verbatim moment exists, drop and re-source.
2. **Novelty-checked** against the named avoid-dup SIOs + the whole corpus (`score-candidate-novelty`).
3. **Magnet-checked** — narrow `key_claim`/keywords; don't reinforce IT-Shankar or ED-Brooks magnets.
4. **Diversity-checked** — no repeated speaker in the batch (max 2 only with reason); ≤8/family;
   TED/TEDx ≤~2 per state; widen gender/culture/field/age/communication style.
5. **Honesty fields** — `transcript_verified: true` and `exact_quote_match` only when truly confirmed.
